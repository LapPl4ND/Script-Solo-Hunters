# 🐉 Solo Hunters Script

A lightweight Roblox script for **Solo Hunters**, designed to automate combat and farming through a simple and intuitive in-game menu.

---

## ✨ Features

- 🎯 Auto Kill
- ⚔️ Auto Skill
- 📍 Teleport to Mobs
- ⚡ Fast and lightweight interface
- 🎮 Easy to use


---

## 📖 Usage

1. Execute the script.
2. Open the menu.
3. Enable **Auto Kill**.
4. Press **TP Mob** once.
5. The script will automatically continue farming nearby enemies.
